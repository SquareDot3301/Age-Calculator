**Simple age calculator in Golang !**

- 1 file
- 0 package

**Installation**

_Step 1 : Clone the repo_

```
git clone github.com/MaelDevFr/Age-Calculator && cd age-calculator-main
```

_Step 2 : Build the repo_

```
make build
```

_Step 3 : Launch the code_

```
./main
```
